# Шаг1: Для запуска проекта необходимо

- полностью скачать проект
  
- запустить терминал
  
- запустить в терминале:
  
  cd <папка в которую быд сохранен проект>
  docker compose down
  docker copmpose up
  
# Шаг2: открыть страницу с отчетом по проекту:

http://localhost:8080/

##  Папки:

- content - в данной папке находятся файлы, необходимые для верстки HTML - страницы, содержащей отчет по сессионному проекту (подход и ход мысли при выполнении проекта)
  
- shared - файлы, которые необходимо было создать в рамках работы над сессионным проектом
